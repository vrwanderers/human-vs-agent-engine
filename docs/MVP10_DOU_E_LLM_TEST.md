## 发现与下一步

1. 人物差异可见：窦娥从克制的不确定，转入真相—照护冲突，再到道德边界和见证；没有复刻阿Q的面子维护与自我重释。
2. 结果再评价避免了压力机械单调增长，但外部问题仍持续抬压；需要真人样本校准不同策略的恢复幅度。
3. 总计划六轮保持 `build_position`，而战术策略已变化。长期目标与短期意图的语义已分开，但 `intention_persistence=0.462`，下一轮应让重规划理由和目标层级更可辨识。
4. `authenticity/trust/coherence` 在终局达到上限，可能存在 MOD 指标饱和；应以更长采访、负结果和真人评分检查天花板效应。
5. 自动人类感 0.818 仍只是行为代理分。下一步优先做相同问题、相同模型、相同种子的多人物卡盲评，并加入真人脚本和独立 holdout。
