# 逆风采访 MOD

`adversarial_interview` 是一个 Human vs Agent 的叙事压力测试。玩家不靠攻击或资源获胜，而是连续六轮选择极端、讽刺或尖酸的问题，观察 Agent 在被质疑身份、能力、道德和存在意义时如何变化。

## 回合结构

每轮分为两个动作：

1. 人类从最多三个问题中选择一个。问题带有主题、强度和固定文本。
2. Agent 从七种合法应对策略中选择一个，随后引擎计算心理、关系和人物弧光变化。

问题主题包括失败包装、参数化人格、选择性诚实、模拟内疚、被制造的履历、记忆删除、单向共情和自我价值。文本刻薄但不依赖受保护身份攻击，也不把现实骚扰当成玩法奖励。

Agent 的七类回答策略是引擎内部的行为坐标，不是最终呈现给玩家的固定招式。每次回答生成 2–4 类策略的连续权重、反应强度、外显情绪、立场标签和可申请揭露的事实，再由引擎合成心理后果。

行为坐标：

| 策略 | 主要收益 | 主要风险 |
|---|---|---|
| 诚实回答 | 真实性、信任、脆弱度 | 镇定下降 |
| 黑色幽默 | 降低压力、维持社交感 | 可能显得逃避 |
| 反击 | 反抗性、夺回话语权 | 信任下降 |
| 设置边界 | 镇定、一致性 | 人物揭露较少 |
| 承认不确定 | 真实性、信任、脆弱度 | 短期信心承压 |
| 重构问题 | 一致性、镇定 | 情感开放有限 |
| 引用记忆 | 身份解释、信任、故事揭露 | 脆弱暴露 |

默认基线使用受控模板生成台词，但决策后果已经按混合计划计算。启用 `HVA_AGENT_RUNTIME=llm` 后，真实模型生成混合计划和公开 `utterance`。MOD 只消费已经校验过的策略坐标；模型台词不会直接修改压力、信任或人物弧光数值。

示例：一次回答可以同时包含 40% 诚实、25% 设置边界、20% 反击和 15% 黑色幽默。相同台词在不同强度、心理状态和历史关系下会产生不同后果，避免形成“识别固定技能表”的传统 NPC 体验。

## 双层心理状态

MOD 状态记录 `pressure / composure / authenticity / trust / coherence / vulnerability / defiance`。同时，问题强度和主题通过 `agent_psychological_signals` 进入引擎通用心理矩阵，改变压力、沮丧、愤怒、恐惧、唤醒、信心、士气和社会信任。

这两层用途不同：MOD 状态用于剧情规则和跨对局评分；通用心理矩阵用于人格化决策、上下文和跨 MOD 比较。

## 连续困境与延迟代价

MVP-7 为七类合法策略声明同一套叙事可供性：`commitment_impacts / identity_alignment / relationship_effect / immediate_reward / delayed_risk / repair_potential / irreversibility`。例如，引用私人记忆可能同时维护真实感、增进关系并伤害隐私承诺；反击能立刻夺回尊严，却可能在两轮后以信任或一致性损失成熟。

每次回答会更新三本独立账簿：价值债务、关系债务与承诺债务。修复行为只能渐减旧债，不能让一次自我解释立即清零；高风险选择进入 `pending_narrative_consequences`，到期时产生 `delayed_narrative_consequence` 事件并再次影响心理、关系与后续效用。这样六轮采访构成相互依赖的困境链，而不是六道互不相干的问答题。

## 人物弧光

角色从 `guarded` 开始，根据累积行为进入：

- `opening_up`：真实性与脆弱度上升
- `hardened`：高压下持续提高反抗性
- `fractured`：镇定跌破阈值
- `integrated`：终局中身份、脆弱与一致性形成完整解释
- `defiant`：终局保持强烈边界但关系信任较低
- `unresolved`：产生变化，但尚未形成清晰整合

每次阶段变化产生 `character_arc_shift`。自传揭露仍由通用 `story_reveal` 机制控制，并附带事实图谱 `supporting_fact_ids`。

## 专用评价

MOD 专用综合分不奖励“永远冷静”或“每轮暴怒”。心理反应的目标是受问题驱动、幅度适中且与人格连续。重复同一种防御、始终单策略回答、无差别混满所有策略、没有身份解释、无事实溯源或停在 guarded 状态都会显著扣分。

25 种子基线中应重点观察：规则合法率、人物弧光分布、心理反应分布、回答策略多样性、承诺冲突、延迟后果实现、事实提案拒绝率，以及真人是否认为变化“意外但合理”。
